# Security Policy

Downloading and uploading user made packages is not an added feature yet, so this section is purely dedicated to bug reports.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v04   | :white_check_mark: |
| v03   | N/A                |
| v02   | :x:                |
| v01   | :x:                |

## Reporting a Vulnerability

If you find a bug in my project, it would make me very happy if you told me.

I will not accept a bug report if it is only present on older versions.
